## ‚ö†Ô∏è Important Notes

1. **Test Mode vs Live Mode:**
   - Use **Test Mode** for development/testing
   - Use **Live Mode** for production
   - Price IDs are different in each mode

2. **Price Format:**
   - In Stripe: Enter as dollars (`29.99`)
   - In Your Form: Enter as cents (`2999`)

3. **One Product, Two Prices:**
   - Create ONE product for each plan
   - Add TWO prices to each product (monthly + yearly)
   - Both prices belong to the same product

4. **Currency:**
   - Use `USD` for US Dollars
   - Change if needed for your region
